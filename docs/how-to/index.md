# How-to guides

Task-focused recipes. Each is self-contained - skim the [Concepts](../concepts/index.md)
first if a term is unfamiliar.

- **[Drive VHI from MyoGestic](drive-from-myogestic.md)** - launch VHI and
  command the control hand over gRPC from Python.
- **[Stream a custom pose](stream-a-custom-pose.md)** - put the control hand
  in `Stream` mode and drive it with an arbitrary continuous 9-DOF pose.
- **[Add a custom movement](add-a-custom-movement.md)** - define a new named
  movement by editing the TOML config.
- **[Build and export](build-and-export.md)** - produce a standalone build,
  including the macOS code-signing fix.
